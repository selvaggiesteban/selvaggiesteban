<h3>1) ¿Qué servicio de OCA se usa para tiendas WooCommerce?</h3>



<p>Para WooCommerce (y para el plugin de <strong>Wanderlust OCA</strong>), el servicio que ofrece OCA es:</p>



<blockquote class="is-layout-flow">
<p>🟣 <strong>OCA e-Pak / OCA Express Pak (servicio para ecommerce)</strong></p>
</blockquote>



<p>Es la plataforma de OCA pensada específicamente para tiendas online: permite gestionar envíos de ecommerce y es la que expone la <strong>API OCA e-Pak</strong> que usa el plugin de Wanderlust para calcular tarifas, sucursales, etc. <a href="https://www.oca.com.ar/Productos?utm_source=chatgpt.com" target="_blank" rel="noreferrer noopener">OCA+1</a></p>



<hr>



<h3>2) ¿Cuál es el formulario de alta de OCA para usar el plugin?</h3>



<p>Para poder vincular tu WordPress + WooCommerce con el plugin de Wanderlust, necesitás:</p>



<ul>
<li><strong>Cuenta de cliente OCA e-Pak / Express Pak</strong>, con:
<ul>
<li>Usuario (email con el que entrás a OCA e-Pak)</li>



<li>Password</li>



<li>N° de cuenta OCA</li>



<li>CUIT y contratos / operativas cargadas (domicilio, sucursal, etc.) <a href="https://shop.wanderlust-webdesign.com/wp-content/uploads/2024/03/Wanderlust-OCA-premium.pdf?utm_source=chatgpt.com" target="_blank" rel="noreferrer noopener">shop.wanderlust-webdesign.com</a></li>
</ul>
</li>
</ul>



<p>El alta se hace así:</p>



<h4>🔹 Opción A – Formulario oficial de OCA Express Pak (recomendado)</h4>



<ol>
<li>Ir al formulario <strong>“OCA EXPRESS PAK – Solicitud de Apertura de Cuenta”</strong>.</li>



<li>Ahí completás datos de tu empresa, ámbito geográfico, web, y seleccionás <strong>“Servicios para E-commerce / Oca Expresspak”</strong>. <a href="https://www1.oca.com.ar/ocaexpresspak/UsuarioWeB/apcuenta.asp?utm_source=chatgpt.com" target="_blank" rel="noreferrer noopener">www1.oca.com.ar+1</a></li>
</ol>



<p>Con eso OCA te contacta y te genera la cuenta de cliente para ecommerce.</p>



<blockquote class="is-layout-flow">
<p>Si querés ubicarlo rápido: buscá en Google<br><strong><code>"OCA EXPRESS PAK - Solicitud de Apertura de Cuenta"</code></strong><br>Es exactamente ese formulario.</p>
</blockquote>



<h4>🔹 Opción B – Contactar a los ejecutivos que recomienda el propio plugin</h4>



<p>En la ficha oficial del plugin <strong>“Wanderlust OCA para WooCommerce”</strong> indican que, para abrir la cuenta con OCA, podés escribir a:</p>



<ul>
<li><strong>Lorena Marienberg – <a>lmarienberg@oca.com.ar</a></strong></li>



<li><strong>Martín Ibarrola – <a>mibarrola@oca.com.ar</a></strong></li>
</ul>



<p>Ellos se encargan de asistirte con la apertura de cuenta para usar la API e-Pak. <a href="https://es-ar.wordpress.org/plugins/wanderlust-oca-e-pak-shipping-rates/?utm_source=chatgpt.com" target="_blank" rel="noreferrer noopener">WordPress.org Español (Argentina)</a></p>



<hr>



<h3>3) Una vez que tengas el alta, ¿qué tenés que cargar en el plugin?</h3>



<p>Con tu cuenta de OCA ya activa y los datos que te envían, en WooCommerce:</p>



<ol>
<li>Instalás y activás <strong>“Wanderlust OCA para WooCommerce”</strong> (o la versión Premium). <a href="https://es-ar.wordpress.org/plugins/wanderlust-oca-e-pak-shipping-rates/?utm_source=chatgpt.com" target="_blank" rel="noreferrer noopener">WordPress.org Español (Argentina)+1</a></li>



<li>En <strong>WooCommerce → Ajustes → Envío → Zonas de envío</strong>, agregás el método <strong>“OCA / Oca Express Pack”</strong>. <a href="https://es-ar.wordpress.org/plugins/wanderlust-oca-e-pak-shipping-rates/?utm_source=chatgpt.com" target="_blank" rel="noreferrer noopener">WordPress.org Español (Argentina)</a></li>



<li>En la configuración del método cargás (según el manual de Wanderlust OCA Premium): <a href="https://shop.wanderlust-webdesign.com/wp-content/uploads/2024/03/Wanderlust-OCA-premium.pdf?utm_source=chatgpt.com" target="_blank" rel="noreferrer noopener">shop.wanderlust-webdesign.com</a>
<ul>
<li>Datos del titular de la cuenta OCA: nombre, apellido, email, dirección, CP, localidad, provincia.</li>



<li><strong>Usuario en OCA</strong> (email de e-Pak).</li>



<li><strong>Password de OCA</strong> (clave de e-Pak).</li>



<li><strong>N° de cuenta OCA</strong> (lo sacás dentro de tu cuenta, en “Condiciones de seguro”).</li>



<li><strong>CUIT</strong> con el formato 00-00000000-0.</li>



<li>Operativas / servicios (envío a domicilio, a sucursal, etc.) que te asignó OCA.</li>
</ul>
</li>



<li>Asegurate de poner <strong>peso y dimensiones</strong> en todos los productos para que calcule tarifas. <a href="https://shop.wanderlust-webdesign.com/wp-content/uploads/2024/03/Wanderlust-OCA-premium.pdf?utm_source=chatgpt.com" target="_blank" rel="noreferrer noopener">shop.wanderlust-webdesign.com+1</a></li>
</ol>
